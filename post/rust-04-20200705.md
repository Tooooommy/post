## fn

```rust
fn _fn(a: usize, b:String) -> String {
    b.repeat(a)
}
```

> fn: 定义声明关键字. 

> _fn: 函数名字

> (a:usize, b:String): 函数参数

> -> String: 返回值.

> b.repeat(a): 不加符号;被当做返回值。 也可以return b.repeat(a);.
